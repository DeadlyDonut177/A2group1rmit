Detailed Description

  • Using the latest Version of “Laragon Server and Laravel” to create and develop the resume builder website on Windows Operating system
  • Making the site as user friendly as possible.
  • Data entered into online form sections are saved into the main database table.
  • Having a choice of how you want your resume to look like, with multiple templates can be utilised to get the desired result to each individual employer
  • To design it in such a way that it can be used in multiple different languages without affecting the template design reaching more people in more countries.
  • Allowing an option to save the resume as a pdf file to personal computer, device or even to the cloud.
  • Optionally this last part can be added to the website to create an income for each resume created, "Create an account", "email and password", "Request for payment screen" to be subscribed to, to receive and download your resume. A subscription can be cancelled if not required after 14 business days so only a one time payment would be required in this case. When the subscription expires or cancels in 14 business days the data can then be automatically wiped from the database so it does not always increasing in size taking up so much storage space on the server hard drive

Technologies Required

An Internet connection, with PC and latest browser installed. A registered email address, Microsoft word and or Acrobat Reader and a Printer. Using latest version of “Laragon”, is being used all over the world to quickly create applications by thousands of developers in windows environments. It is highly customizable to use in PHP and Mysql projects. Utilising “Laravel 8” a web application framework with expressive, elegant syntax. I believe that using Laravel 8 to be the best choice for building a modern full-stack web application.

An income could also be generated from a completed website in multiple countries around the globe with the additional languages added to the database.
